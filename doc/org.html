Installation

Installing required packages:


sudo apt-get install nodejs ↵

Required libarys are in the repository, and were installed with the following commands:

sudo apt-get install node-ws node-connect node-serve-static ↵

or use npm to install nodejs packages:

sudo apt-get install npm ↵
npm install ws ↵
npm install connect ↵
npm install serve-static ↵

Starting the server:


cd server ↵
nodejs server.js ↵

Starting the client:


http://localhost:1337/client/view.html
